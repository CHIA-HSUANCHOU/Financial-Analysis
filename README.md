## Description
選擇29檔科技電子股，先分析個別資產累積報酬率、敘述統計量
使用
1. Buy and hold：一開始均等比例投資，不進行任何買賣直到投資結束。 
2. Equal weight：每期調整投資組合，讓各個股票投資比例相同。 
3. Price weight。 
4. GMVP。 
5. No-shortsales GMVP。 
6. Tangency portfolio。 
7. No-shortsales MVP：期望報酬是0.3%
